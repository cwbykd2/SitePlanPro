# Project Summary
SitePlan Pro is an advanced interactive site planning tool designed for urban planners and developers. It enables users to visualize and organize urban elements, such as buildings, roads, and ponds, using satellite imagery maps. The application enhances decision-making in urban development with features like real-time area statistics, customizable site boundaries, cost estimation tools for construction, and a robust site management system for saving and switching between multiple site plans. Recent updates have improved the functionality of the Sites dropdown button in published versions.

# Project Module Description
- **Map & Drawing**: Interactive Leaflet map for drawing and editing urban elements.
- **Building Configuration**: Configurable building types with adjustable settings.
- **Statistics**: Real-time area calculations for various site elements.
- **Geocoding**: Address search and reverse geocoding functionalities.
- **Editable Site Boundary**: Interactive modification of site boundaries.
- **Building Manipulation**: Enhanced interactivity for managing buildings.
- **Real-Time Dimension Labels**: Live dimension displays during modifications.
- **Paving Tool**: Select paving types (Car Parking, Trailer Parking, Other) with corresponding colors and labels displayed on the map.
- **Trailer Parking Configuration**: Options for trailer parking layouts.
- **Site Management**: Features for naming, auto-saving changes, and exporting site plans, including a dropdown to manage saved sites and options to create new site plans.
- **Parking Depths**: Default settings for parking depths.
- **Truck Court Placement**: Aligns truck courts with building edges.
- **Boundary Locking**: Visual confirmation of site boundaries.
- **Road Drawing Enhancement**: Improved road drawing capabilities.
- **Curved Roads**: Creation of smooth curved road segments.
- **Fee Estimator**: Input, save, and load construction fee templates with auto-calculation.
- **Truck Court Inset**: Simplified input for truck court inset distances.
- **Building Cost Estimator**: Calculates costs based on individual building specifications.
- **Paving Interaction**: Enables resizing, dragging, and rotating of paving areas, mirroring building functionalities.

# File Description Inventory
- **index.html**: Contains the user interface, including editable boundaries, building manipulation, paving type selection, cost estimation tools, and site management functionalities.
- **script.js**: Implements application logic and interactive features, including paving interactions and functionalities.
- **style.css**: Provides styling for the application and its elements.
- **package.json**: Manages project dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API, Nominatim
- **PDF Generation**: html2canvas, jsPDF
- **Fonts**: Google Fonts (DM Mono, Syne)

# Usage
To set up the project:
1. Install dependencies from `package.json`.
2. Build the project as needed.
3. Run the application using the provided development server script.
