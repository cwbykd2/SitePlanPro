# Project Summary
SitePlan Pro is an advanced interactive site planning tool tailored for urban planners and developers. It empowers users to visualize and organize buildings, roads, and ponds on satellite imagery maps, enhancing decision-making in urban development. The tool features real-time area statistics, customizable site boundaries, and the capability to generate downloadable PDF documents, streamlining the planning process with dynamic data visualization and interactivity. Recent enhancements include a road tie-in feature that allows roads to connect seamlessly to truck courts, car parking areas, and paving. The latest updates have removed the road snapping functionality, allowing for more manual control in road editing.

# Project Module Description
- **Map & Drawing**: Interactive Leaflet map for drawing and editing buildings, roads, ponds, and paving areas.
- **Building Configuration**: Configurable building types with adjustable bay spacing.
- **Statistics**: Real-time calculations for various areas, including building and paved surface area.
- **Geocoding**: Address search functionality using the Photon API.
- **Editable Site Boundary**: Users can modify site boundaries interactively.
- **Building Manipulation**: Enhanced interactivity for dragging, rotating, and resizing buildings using mouse handles.
- **Real-Time Dimension Labels**: Displays live dimensions during manipulation.
- **Paving Tool**: Add paving areas with real-time dimension display.
- **Trailer Parking Configuration**: Options for configuring trailer parking.
- **Site Management**: Features for naming sites, auto-saving changes, and exporting site plans as PDFs.
- **Parking Depths**: Default parking depth settings.
- **Truck Court Placement**: Aligns truck court and parking with building edges.
- **Boundary Locking**: Confirms site boundary with a hashed line.
- **Adjust Site Button**: Re-enables editing of the site boundary.
- **Out-of-Boundary Highlighting**: Highlights features extending outside the site boundary.
- **Road Drawing Enhancement**: Users can now press Enter to finish drawing a road at the current point, alongside the existing double-click method.
- **Road Width Selection**: After drawing a road, users can select from preset widths (30', 36', 40') or enter a custom width, with roads rendered as grey polygons.
- **Curved Roads**: Users can create smooth curved segments using a Catmull-Rom spline toggle.
- **Road Editing**: Click roads to enter edit mode, allowing reshaping and vertex management.
- **Road Intersections**: Automatically detects and merges overlapping road polygons.

# File Description Inventory
- **index.html**: User interface with editable boundaries, building manipulation, real-time labels, paving tool, site naming, auto-save indicators, road width modal, and PDF export functionality.
- **script.js**: Implements application logic, including real-time updates, interactive features, and the new road functionalities.
- **style.css**: Provides styles for the application and its elements, including the road editing indicators.
- **package.json**: Manages dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API
- **PDF Generation**: html2canvas, jsPDF
- **Fonts**: Google Fonts (DM Mono, Syne)
