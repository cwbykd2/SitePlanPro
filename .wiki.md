# Project Summary
SitePlan Pro is an advanced interactive site planning tool designed for urban planners and developers. It enables users to visualize and layout buildings, roads, and ponds on satellite imagery maps. By leveraging Leaflet.js, the application enhances decision-making in urban planning through real-time area statistics, address geocoding, editable site boundaries, and advanced building manipulation tools. The platform's dynamic data visualization and interactive features make it invaluable for effective urban development.

# Project Module Description
- **Map & Drawing**: Users can interact with a Leaflet map to draw and edit buildings, roads, ponds, and new paving areas.
- **Building Configuration**: Options for building types (cross dock/rear load) and bay spacing.
- **Statistics**: Real-time calculations of building area, paved surface area, pond area, and new paving area.
- **Geocoding**: Address search functionality using the Photon API.
- **Editable Site Boundary**: Users can draw and modify site boundaries by dragging corners and adding points.
- **Building Manipulation**: Users can drag, rotate, and resize buildings with enhanced interactivity through new corner handles.
- **Real-Time Dimension Labels**: Displays real-time width, depth, and square footage of buildings and paving during manipulation.
- **Paving Tool**: Allows users to add paving areas with options for rectangle drag or free draw, displaying dimensions in real-time.
- **Trailer Parking Configuration**: New options for cross dock and rear load buildings allow users to configure trailer parking on both sides, adjusting truck court dimensions based on user input.

# File Description Inventory
- **index.html**: Contains the complete user interface, including features for editable site boundaries, building manipulation, real-time dimension labels, the new paving tool, and trailer parking options.
- **script.js**: Integrated JavaScript logic for application functionality, including real-time updates for building and paving dimensions, and new interactive handles.
- **style.css**: Integrated styles for the application, including styles for dimension labels and the new paving tool.
- **package.json**: Configuration file for managing dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API
- **Fonts**: Google Fonts (DM Mono, Syne)
