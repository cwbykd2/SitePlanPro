# Project Summary
SitePlan Pro is an advanced interactive site planning tool designed for urban planners and developers. It enables users to visualize and arrange urban elements such as buildings, roads, and ponds using satellite imagery, enhancing decision-making in urban development. The application features real-time area statistics, customizable site boundaries, cost estimation tools, and a comprehensive site management system. Recent updates have improved user interaction, particularly with parking configurations and road editing, alongside various functionality enhancements and bug fixes.

# Project Module Description
- **Map & Drawing**: Interactive Leaflet map for drawing and editing urban elements.
- **Building Configuration**: Configurable building types with an optimized Bay Configuration system.
- **Statistics**: Real-time area calculations for various site elements, including a new "Car Parks" section that counts parking spots around each building.
- **Geocoding**: Address search and reverse geocoding functionalities.
- **Editable Site Boundary**: Interactive modification of site boundaries.
- **Building Manipulation**: Enhanced interactivity for managing buildings.
- **Real-Time Dimension Labels**: Live dimension displays during modifications.
- **Paving Tool**: Select paving types with corresponding colors and labels.
- **Trailer Parking Configuration**: Options for trailer parking layouts.
- **Site Management**: Features for naming, auto-saving changes, and exporting site plans.
- **Parking Depths**: Default settings for parking depths with enhanced label visibility.
- **Truck Court Placement**: Aligns truck courts with building edges.
- **Boundary Locking**: Visual confirmation of site boundaries.
- **Road Drawing Enhancement**: Improved road drawing capabilities, including draggable midpoints and auto-edit mode after placement.
- **Curved Roads**: Creation of smooth curved road segments.
- **Fee Estimator**: Input, save, and load construction fee templates.
- **Building Cost Estimator**: Calculates costs based on building specifications, including updated square footage prices.
- **Bump-Out Feature**: Enhanced design flexibility for building layouts.
- **Depth Labels**: New depth labels for car parking and truck court overlays.
- **Parking Overlay Sub-Zones**: Detailed delineation for parking areas, including sidewalks, car stalls, and fire lanes with distinct visual patterns.
- **PDF Export**: Exports only the map view without headers, statistics, or costs, centering the map on the page.
- **Parking Configuration Update**: Replaced Double/Single toggles with numeric row inputs (1-4) for parking on all sides, enhancing layout flexibility.
- **Bay Width Update**: Added a 60' option to the Bay Width selector, enhancing configuration options for users.
- **Instant Parking Rows**: Parking rows now appear immediately upon selecting parking options, improving user experience.
- **Paving Label Update**: Paving labels are now displayed in black text with a white border for better readability.
- **Car Parking Pattern**: New functionality to fill paving areas labeled as car parking with a specific pattern of stalls and fire lanes, enhancing visual consistency and usability.
- **Fire Lane Update**: Removed red diagonal hash marks from fire lanes and changed borders to solid red (1px thick).
- **Additional Parking Feature**: New functionality to display car parking added from the paving tab as "Additional Parking" under the CAR PARKS section, with individual lot counts included in the total.

# File Description Inventory
- **index.html**: Contains the user interface, including editable boundaries, building manipulation, paving type selection, cost estimation tools, and updated visibility for building dimension labels. Recently updated to include a "Car Parks" section that counts and displays the number of car parking spots around each building, now also showing "Additional Parking" from the paving tab.
- **script.js**: Implements application logic and interactive features, including paving interactions and building management.
- **style.css**: Provides styling for the application and its elements.
- **package.json**: Manages project dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API, Nominatim
- **PDF Generation**: html2canvas, jsPDF
- **Fonts**: Google Fonts (DM Mono, Syne)

# Usage
To set up the project:
1. Install dependencies from `package.json`.
2. Build the project as needed.
3. Run the application using the provided development server script.
