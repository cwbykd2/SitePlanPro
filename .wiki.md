# Project Summary
SitePlan Pro is an interactive site planning tool that enables users to layout buildings, roads, and ponds on satellite imagery maps. The application employs Leaflet.js for mapping and drawing, allowing urban planners and developers to visualize site designs and compute area statistics in real-time. It includes features like address geocoding, editable site boundaries, and advanced building manipulation tools, making it an essential resource for effective urban planning.

# Project Module Description
- **Map & Drawing**: Users can interact with a Leaflet map to draw buildings, roads, and ponds.
- **Building Configuration**: Options for building types (cross dock/rear load) and bay spacing.
- **Statistics**: Real-time calculations of building area, paved surface area, pond area, and site coverage percentage.
- **Geocoding**: Address search functionality using the Photon API.
- **Editable Site Boundary**: Users can draw and edit site boundaries by dragging corners and adding points to refine the parcel shape.
- **Building Manipulation**: Users can drag, rotate, and resize buildings after placement, enhancing interactivity.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API
- **Fonts**: Google Fonts (DM Mono, Syne)
