# Project Summary
SitePlan Pro is an advanced interactive site planning tool designed for urban planners and developers. It enables users to visualize and organize urban elements—such as buildings, roads, and ponds—using satellite imagery. The application enhances decision-making in urban development with features like real-time area statistics, customizable site boundaries, cost estimation tools, and a comprehensive site management system. Recent updates include improved car parking configurations, allowing for customizable parking depths for better usability in site planning.

# Project Module Description
- **Map & Drawing**: Interactive Leaflet map for drawing and editing urban elements.
- **Building Configuration**: Configurable building types with an optimized Bay Configuration system showing bay grids.
- **Statistics**: Real-time area calculations for various site elements, including bump-outs.
- **Geocoding**: Address search and reverse geocoding functionalities.
- **Editable Site Boundary**: Interactive modification of site boundaries.
- **Building Manipulation**: Enhanced interactivity for managing buildings.
- **Real-Time Dimension Labels**: Live dimension displays during modifications, improved for readability.
- **Paving Tool**: Select paving types with corresponding colors and labels displayed on the map.
- **Trailer Parking Configuration**: Options for trailer parking layouts.
- **Site Management**: Features for naming, auto-saving changes, and exporting site plans.
- **Parking Depths**: Default settings for parking depths with enhanced label visibility.
- **Truck Court Placement**: Aligns truck courts with building edges, auto-inset with bump outs.
- **Boundary Locking**: Visual confirmation of site boundaries.
- **Road Drawing Enhancement**: Improved road drawing capabilities.
- **Curved Roads**: Creation of smooth curved road segments.
- **Fee Estimator**: Input, save, and load construction fee templates with auto-calculation.
- **Building Cost Estimator**: Calculates costs based on individual building specifications, now includes bump-out square footage.
- **Paving Interaction**: Enables resizing, dragging, and rotating of paving areas, mirroring building functionalities.
- **Bump-Out Feature**: Enhanced design flexibility with new options for cross dock and rear load buildings, including color matching and dimension labeling.
- **Depth Labels**: New depth labels for car parking and truck court overlays, now with improved styling.

# File Description Inventory
- **index.html**: Contains the user interface, including editable boundaries, building manipulation, paving type selection, cost estimation tools, and updated visibility for building dimension labels.
- **script.js**: Implements application logic and interactive features, including paving interactions and building management.
- **style.css**: Provides styling for the application and its elements.
- **package.json**: Manages project dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API, Nominatim
- **PDF Generation**: html2canvas, jsPDF
- **Fonts**: Google Fonts (DM Mono, Syne)

# Usage
To set up the project:
1. Install dependencies from `package.json`.
2. Build the project as needed.
3. Run the application using the provided development server script.
