# Project Summary
SitePlan Pro is an advanced interactive site planning tool tailored for urban planners and developers. It empowers users to visualize and layout buildings, roads, and ponds on satellite imagery maps, significantly enhancing decision-making in urban planning. The platform features real-time area statistics, address geocoding, editable site boundaries, advanced building manipulation tools, and the ability to generate downloadable PDF documents of site plans. By facilitating dynamic data visualization and interactivity, SitePlan Pro supports effective urban development.

# Project Module Description
- **Map & Drawing**: Interactive Leaflet map for drawing and editing buildings, roads, ponds, and paving areas.
- **Building Configuration**: Configurable building types (cross dock/rear load) and bay spacing.
- **Statistics**: Real-time calculations for building area, paved surface area, pond area, and paving area.
- **Geocoding**: Address search functionality using the Photon API.
- **Editable Site Boundary**: Users can modify site boundaries interactively.
- **Building Manipulation**: Enhanced interactivity for dragging, rotating, and resizing buildings.
- **Real-Time Dimension Labels**: Displays live dimensions of buildings and paving during manipulation.
- **Paving Tool**: Add paving areas with real-time dimension display.
- **Trailer Parking Configuration**: Options for configuring trailer parking on both sides of buildings.
- **Site Management**: Features for naming sites upon creation, auto-saving changes, managing saved sites, and exporting site plans as PDFs.
- **Parking Depths**: Default parking depth for left/right side is 62 ft; truck court depths vary based on configuration.
- **Truck Court Placement**: For rear load buildings, the truck court is now automatically placed on the longer side, ensuring optimal layout.

# Directory Tree

# File Description Inventory
- **index.html**: Contains the user interface, including features for editable boundaries, building manipulation, real-time labels, paving tool, site naming, auto-save indicators, and a button for exporting the site plan as a PDF.
- **script.js**: Implements application logic, including real-time updates, interactive features, site management functionalities, and the PDF export function.
- **style.css**: Provides styles for the application, including dimension labels, the paving tool, and site management elements.
- **package.json**: Manages dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API
- **PDF Generation**: html2canvas, jsPDF
- **Fonts**: Google Fonts (DM Mono, Syne)

# Usage
To set up the project:
