# Project Summary
SitePlan Pro is an advanced interactive site planning tool designed for urban planners and developers. It enables users to visualize and layout buildings, roads, and ponds on satellite imagery maps, enhancing decision-making in urban planning. Key features include real-time area statistics, address geocoding, editable site boundaries, and advanced building manipulation tools. Its dynamic data visualization and interactivity are essential for effective urban development, with new functionalities that support site naming and auto-saving.

# Project Module Description
- **Map & Drawing**: Interactive Leaflet map for drawing and editing buildings, roads, ponds, and paving areas.
- **Building Configuration**: Configurable building types (cross dock/rear load) and bay spacing.
- **Statistics**: Real-time calculations for building area, paved surface area, pond area, and paving area.
- **Geocoding**: Address search functionality using the Photon API.
- **Editable Site Boundary**: Users can modify site boundaries interactively.
- **Building Manipulation**: Enhanced interactivity for dragging, rotating, and resizing buildings.
- **Real-Time Dimension Labels**: Displays live dimensions of buildings and paving during manipulation.
- **Paving Tool**: Add paving areas with real-time dimension display.
- **Trailer Parking Configuration**: Options for configuring trailer parking on both sides of buildings.
- **Site Management**: New features for naming sites upon creation, auto-saving changes, and managing saved sites through a dropdown interface.

# File Description Inventory
- **index.html**: Contains the user interface, including features for editable boundaries, building manipulation, real-time labels, paving tool, site naming, and auto-save indicators.
- **script.js**: Implements application logic, including real-time updates, interactive features, and site management functionalities.
- **style.css**: Provides styles for the application, including dimension labels, the paving tool, and new site management elements.
- **package.json**: Manages dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API
- **Fonts**: Google Fonts (DM Mono, Syne)
