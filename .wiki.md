# Project Summary
SitePlan Pro is an interactive site planning tool designed to facilitate the layout of buildings, roads, and ponds on satellite imagery maps. Leveraging Leaflet.js for mapping and drawing functionalities, it allows users to visualize their site designs and calculate area statistics in real-time. The application supports address geocoding and provides various tools for detailed site planning.

# Project Module Description
- **Map & Drawing**: Users can interact with a Leaflet map to draw buildings, roads, and ponds.
- **Building Configuration**: Options for building types (cross dock/rear load) and bay spacing.
- **Statistics**: Real-time calculations of building area, paved surface area, pond area, and site coverage percentage.
- **Geocoding**: Address search functionality using the Photon API.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API
- **Fonts**: Google Fonts (DM Mono, Syne)

# Usage
To set up the project:
