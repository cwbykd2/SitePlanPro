# Project Summary
SitePlan Pro is an advanced interactive site planning tool aimed at urban planners and developers. It allows users to visualize and organize urban elements—such as buildings, roads, and ponds—using satellite imagery. The application offers real-time area statistics, customizable site boundaries, cost estimation tools, and comprehensive site management features. Recent updates have focused on enhancing layout configurations and user interactions, particularly improving visibility and usability.

# Project Module Description
- **Map & Drawing**: Interactive Leaflet map for drawing and editing urban elements.
- **Building Configuration**: Configurable building types with an optimized Bay Configuration system.
- **Statistics**: Real-time area calculations for various site elements, including a "Car Parks" section that counts parking spots around each building.
- **Geocoding**: Address search and reverse geocoding functionalities.
- **Editable Site Boundary**: Interactive modification of site boundaries.
- **Building Manipulation**: Enhanced interactivity for managing buildings, including real-time bay calculations.
- **Real-Time Dimension Labels**: Live dimension displays during modifications.
- **Paving Tool**: Select paving types with corresponding colors and labels.
- **Trailer Parking Configuration**: Options for trailer parking layouts.
- **Site Management**: Features for naming, auto-saving changes, and exporting site plans.
- **Parking Depths**: Default settings for parking depths with enhanced label visibility.
- **Truck Court Placement**: Aligns truck courts with building edges.
- **Boundary Locking**: Visual confirmation of site boundaries.
- **Road Drawing Enhancement**: Improved road drawing capabilities, including draggable midpoints and auto-edit mode after placement.
- **Curved Roads**: Creation of smooth curved road segments.
- **Fee Estimator**: Input, save, and load construction fee templates.
- **Building Cost Estimator**: Calculates costs based on building specifications, including updated square footage prices.
- **Bump-Out Feature**: Enhanced design flexibility for building layouts, with draggable edges replaced by custom width and depth inputs for bumpouts.
- **Depth Labels**: New depth labels for car parking and truck court overlays.
- **Parking Overlay Sub-Zones**: Detailed delineation for parking areas, including sidewalks, car stalls, and fire lanes.
- **PDF Export**: Exports only the map view without headers, centering the map on the page.
- **Parking Configuration Update**: Enhanced layout flexibility with numeric row inputs for parking.
- **Instant Parking Rows**: Parking rows now appear immediately upon selecting parking options.
- **Paving Label Update**: Improved readability of paving labels.
- **Car Parking Pattern**: New functionality for visual consistency in paving areas labeled as car parking.
- **Fire Lane Update**: Improved visibility for fire lanes with solid borders.
- **Stall Count Font Size Update**: Improved readability of stall count numbers.
- **Zoom-Responsive Label Sizing**: Enhanced visibility of building footprint during zoom.
- **Overlay Visibility Fix**: Improved focus during building customization.
- **Bay Size Calculation Fix**: Updated logic for consistent bay sizes during building customization.
- **Building Rotation Bug Fix**: Resolved issue where changing bay width or speed caused the building to flip unexpectedly.
- **Truck Court Positioning Fix**: Corrected the alignment of truck courts for cross dock buildings.
- **Inward Bumpout Dragging**: Enabled inward dragging of bumpouts up to 200 ft into buildings.
- **Map Layout Fix**: Resolved the issue of the map overlaying the sidebar by adjusting CSS properties for proper display.

# File Description Inventory
- **index.html**: Contains the user interface, including editable boundaries, building manipulation, paving type selection, cost estimation tools, and updated visibility for building dimension labels. Recently updated to fix map layout issues by adjusting CSS properties.
- **script.js**: Implements application logic and interactive features, including paving interactions and building management.
- **style.css**: Provides styling for the application and its elements.
- **package.json**: Manages project dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API, Nominatim
- **PDF Generation**: html2canvas, jsPDF
- **Fonts**: Google Fonts (DM Mono, Syne)

# Usage
To set up the project:
1. Install dependencies from `package.json`.
2. Build the project as needed.
3. Run the application using the provided development server script.
