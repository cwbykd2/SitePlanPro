# Project Summary
SitePlan Pro is an advanced interactive site planning tool tailored for urban planners and developers. It enables users to visualize and layout buildings, roads, and ponds on satellite imagery maps. By leveraging Leaflet.js, the application offers real-time area statistics, address geocoding, editable site boundaries, and advanced building manipulation tools. This tool enhances decision-making processes in urban planning and development by providing dynamic data visualization and interactive features.

# Project Module Description
- **Map & Drawing**: Users can interact with a Leaflet map to draw and edit buildings, roads, and ponds.
- **Building Configuration**: Options for building types (cross dock/rear load) and bay spacing.
- **Statistics**: Real-time calculations of building area, paved surface area, pond area, and site coverage percentage.
- **Geocoding**: Address search functionality using the Photon API.
- **Editable Site Boundary**: Users can draw and modify site boundaries by dragging corners and adding points.
- **Building Manipulation**: Users can drag, rotate, and resize buildings, enhancing interactivity with new corner handles.
- **Real-Time Dimension Labels**: Displays real-time width, depth, and square footage of buildings during manipulation.

# File Description Inventory
- **index.html**: Contains the complete user interface, including features for editable site boundaries, building manipulation, and real-time dimension labels.
- **script.js**: Integrated JavaScript logic for application functionality, including real-time updates for building dimensions and new interactive handles.
- **style.css**: Integrated styles for the application, including styles for dimension labels.
- **package.json**: Configuration file for managing dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API
- **Fonts**: Google Fonts (DM Mono, Syne)
