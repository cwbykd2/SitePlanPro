# Project Summary
SitePlan Pro is an advanced interactive site planning tool designed for urban planners and developers. It enables users to visualize and organize urban elements such as buildings, roads, and ponds on satellite imagery maps, facilitating informed decision-making in urban development. Key features include real-time area statistics, customizable site boundaries, and downloadable PDF documents. Recent enhancements have introduced a Fee Estimator panel for estimating construction fees, and a new Truck Court Inset feature, which now allows users to specify inset distances for truck courts with simplified input.

# Project Module Description
- **Map & Drawing**: Interactive Leaflet map for drawing and editing buildings, roads, ponds, and paving areas.
- **Building Configuration**: Configurable building types with adjustable bay spacing.
- **Statistics**: Real-time calculations for areas, including paved areas and ponds.
- **Geocoding**: Address search functionality using the Photon API and reverse geocoding for city detection via Nominatim.
- **Editable Site Boundary**: Users can modify site boundaries interactively.
- **Building Manipulation**: Enhanced interactivity for dragging, rotating, and resizing buildings.
- **Real-Time Dimension Labels**: Live dimensions displayed during manipulation, currently focusing on road widths.
- **Paving Tool**: Add paving areas with real-time dimension display.
- **Trailer Parking Configuration**: Options for configuring trailer parking.
- **Site Management**: Features for naming sites, auto-saving changes, and exporting site plans as PDFs.
- **Parking Depths**: Default parking depth settings.
- **Truck Court Placement**: Aligns truck court and parking with building edges.
- **Boundary Locking**: Confirms site boundary visually.
- **Adjust Site Button**: Re-enables editing of the site boundary.
- **Out-of-Boundary Highlighting**: Highlights features that extend outside the site boundary.
- **Road Drawing Enhancement**: Users can finish drawing a road by pressing Enter or double-clicking.
- **Road Width Selection**: Users can select from preset or custom road widths.
- **Curved Roads**: Create smooth curved segments using a Catmull-Rom spline toggle.
- **Road Editing**: Click roads to enter edit mode for reshaping and vertex management.
- **Road Intersections**: Automatically detects and merges overlapping road polygons.
- **Pond Drawing Enhancement**: Ponds now display square footage labels at their center.
- **Fee Estimator**: A panel that allows users to input, save, and load construction fee templates, auto-calculate totals, and includes fee summaries in PDF exports.
- **Truck Court Inset**: Users can specify inset distances for truck courts with simplified input for top/bottom sides, enhancing layout flexibility.

# File Description Inventory
- **index.html**: User interface file that includes features for editable boundaries, building manipulation, real-time labels (now showing only road widths), paving tool, site naming, auto-save indicators, updated pond drawing functionality with square footage labels, the newly integrated Fee Estimator panel, and the simplified Truck Court Inset feature.
- **script.js**: Implements application logic, including real-time updates and interactive features, along with the new fee estimator and simplified truck court inset functionalities.
- **style.css**: Provides styles for the application and its elements.
- **package.json**: Manages project dependencies and scripts for development.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Mapping**: Leaflet.js (v1.9.4)
- **Drawing Tools**: Leaflet.draw (v1.0.4)
- **Geospatial Calculations**: Turf.js (v6.5.0)
- **Geocoding**: Photon API, Nominatim
- **PDF Generation**: html2canvas, jsPDF
- **Fonts**: Google Fonts (DM Mono, Syne)

# Usage
To set up the project:
1. Install dependencies from `package.json`.
2. Build the project as needed.
